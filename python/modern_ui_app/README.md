# Modern UI Todo App

这是一个使用 Python 和 [Flet](https://flet.dev) 构建的现代化待办事项应用。

## 特性

- ✨ **美观的 UI**：Material Design 3 风格，圆角、阴影和动效。
- 🌓 **主题切换**：支持深色/浅色模式一键切换。
- 📱 **响应式**：适配桌面和移动端布局。
- 📝 **功能完整**：添加、编辑、删除、筛选（全部/进行中/已完成）。

## 如何运行

1. 确保已创建虚拟环境并安装依赖（如果尚未安装）：
   ```powershell
   python -m venv .venv
   .venv\Scripts\pip install flet
   ```

2. 运行应用：
   ```powershell
   .venv\Scripts\python main.py
   ```

## 依赖

- Python 3.7+
- Flet
